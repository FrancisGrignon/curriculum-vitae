2014-2015

Consumer Price Index System

Statistics Canada

Statistics Canada was working on a new system to replace the old one that was calculating the Consumer Price Index (CPI). That new system had major performance issues.

My mandate was to pinpoint the cause of performance issue and resolve it.

I first ask for a demo of the product to know where were the issues. Then I did a code review to see pinpoint the defect code. It was due to as web service which was transfering to much data. I did clean to reduce the data transfered. I invented a invovative method then I called circular object streaming.

The time to load the form when down from 10 minutes to less then 10 seconds for the same amount of relevant data.
