2014-2015

Consumer Price Index System

Statistics Canada

Statistics Canada was working on a new system to replace the old one that was calculating the Consumer Price Index (CPI). That new system had major performance issues .

My mandate was to pinpoint the cause of performance issues and resolve them.

I first asked for a demo of the product to know where the issues were. Then, I did a code review to pinpoint the defect code. It was due to a web service which was transferring too much data. I cleaned it up to reduce the data transferred. I invented an innovative method that I called circular object streaming.

The time to load the form went down from 10 minutes to less then 10 seconds for the same amount of relevant data.

2013-2014

Integrated Collection and Operation Systems (ICOS)

Statistics Canada

My task was to build (code and test) Java web services for the 2015 Census.

I proposed creating a standard framework for the web services based on improving the traceability and the reusability. My suggestion was approved. I built framework using abstract classes and generic types.

I added a traceability mechanism to help trace execution errors between the web service client and the server. It helped pinpoint errors.

My major contribution helped the team to develop standard web services in less time.

2011-2013

Census and electronic questionnaires (EQ)

Statistics Canada

EQ allows respondents to file surveys online. At that time, it was necessary to improve its capacity to 55K simultaneous users to be ready for the 2016 census.

  • I did code review to better understand the code and to pinpoint code issues.
  • I did different load tests, integration tests and unit tests.
  • I traced the code execution to better understand the bottlenecks and to fix them.

The system used to generate proof of statement was not supported anymore by the provided. The City had to build a new one.

2010-2011

Corporate services

City of Gatineau

The system used to generate proof of statement was not supported anymore by the provided. The City had to build a new one.

I was asked to build a new system to replace it.

From the Government Tax specifications, I wrote the specifications for the new system and I designed the architecture of the system including the database model. I helped the team with the coding of the system.
